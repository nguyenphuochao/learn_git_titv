- git reset --soft <commit id>
    Di chuyển HEAD về vị trí commit
    Trạng thái của stage và tất cả sự thay đổi của file được giữ nguyên
- git reset <commit id>
    Di chuyển HEAD về vị trí commit reset
    Vẫn giữ tất cả thay đổi của file, nhưng loại bỏ các thay đổi stage
    (--mixed)
- git reset --hard <commit id>
    Di chuyển con trỏ HEAD về vị trí commit reset và loại bỏ tất cả 
    sự thay đổi của file, stage
